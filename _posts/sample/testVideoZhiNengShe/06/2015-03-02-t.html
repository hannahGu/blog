            基本数据类型：number string boolean function object undefined(有定义未赋值,未定义)（null空值）
            js组成：ECMAScript DOM Browser
                ECMAScript：翻译，是解释器，ECMAScript 是一种开放的、国际上广为接受的脚本语言规范。
                JavaScript 是 ECMAScript 规范的一种实现。 
                DOM：Document Object Model,操作html的能力， document
                BOM：Browser.........
            类型转换：
                显示转换，强制转换
                    parseInt不仅可以str to num，而且去掉字符“12px34”=》12，从左往右看，直到遇到非数字跳出
                    NaN ；not a number，NaN 不等于NaN，判断时候用isNaN（）来看结果
                隐式类型转换
                    ==先转换类型，然后比较
                    ===不转换，直接比
                    -减法， / *
            闭包：子函数可以使用父函数的局部变量
            命名：

                整数：i

                正则：re
