                二级菜单在一级菜单的相应的子元素下面
                    1.位置问题不用担心（偏移省了）
                    2.二级菜单要加position absolute(飞到空中去，不占位置？！)
                    3.css中加hover，省了写mouseout这样的
                    4.直接子元素的调用方法

                #nav{
                    margin:0 auto;
                    width:400px;
                }
                #nav div{
                    display: inline-block;
                }
                #nav ul{
                    list-style: none;
                }
                #nav>div>ul{
                    padding:0;
                    margin:0;
                    display:none;
                    position: absolute;
                }
                #nav>div:hover>ul{
                    display:block;
                }
